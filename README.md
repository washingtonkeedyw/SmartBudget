# SmartBudget

[![License](https://img.shields.io/badge/license-MIT-green.svg)](LICENSE)

## Overview
SmartBudget is a modern, intuitive personal finance tracker designed to help users effortlessly manage their monthly budget, track expenses, and visualize spending habits with insightful charts. Built with simplicity and efficiency in mind, SmartBudget empowers you to take control of your finances.

## Features
- Add, edit, and delete income and expense entries
- Categorize transactions for detailed reporting
- Interactive charts and graphs for spending insights
- Monthly budget goals with real-time tracking
- Export data as CSV

## Tech Stack
- React.js for dynamic user interface
- Node.js and Express backend API
- MongoDB for data storage
- Chart.js for data visualization

## Installation
1. Clone the repository:
   ```
   git clone https://github.com/washingtonkeedyw/SmartBudget.git
   ```
2. Navigate to the project folder:
   ```
   cd SmartBudget
   ```
3. Install dependencies for frontend and backend:
   ```
   cd client && npm install
   cd ../server && npm install
   ```
4. Start backend server:
   ```
   npm run start
   ```
5. Start frontend application:
   ```
   cd ../client && npm start
   ```

## Usage
Once the app is running, open your browser and navigate to http://localhost:3000. Sign up or log in to start tracking your budget. Use the dashboard to add transactions and visualize your spending patterns. Export your data anytime for offline analysis.
